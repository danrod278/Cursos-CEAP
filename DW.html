Desenvolvimento Web


O Desenvolvimento Web refere-se ao processo de criação e manutenção de sites e aplicativos web. Envolve uma variedade de habilidades e disciplinas para produzir uma experiência interativa e funcional para os usuários na internet. Durante os quatro semestres, os alunos irão se aprofundar nessa área vendo diversos conteúdos que formam um Site completo para a WEB. Os principais conteudos aprendidos são: HTML e CSS, JavaScript e Frameworks.
