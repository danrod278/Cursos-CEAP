Cloud Computing


Cloud Computing, ou computação em nuvem, é um modelo de entrega de serviços de computação pela internet. Os alunos são apresentados aos conceitos inicais da computação em nuvem, incluindo serviços, modelos de implantação e características essenciais. Além de ganharem experiência prática com plataformas de nuvem populares, como Amazon Web Services (AWS), Microsoft Azure, Google Cloud Platform (GCP), IBM Cloud, entre outras.
