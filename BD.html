Banco de Dados


Os alunos aprendem sobre a organização, o design e a manipulação eficaz de dados. A disciplina abrange uma variedade de tópicos, desde conceitos fundamentais até técnicas avançadas. O aluno aprende o que é um banco de dados e suas vantagens e aplicações. Outro tópico essencial da matéria, é a Introdução a sistemas como MySQL, PostgreSQL, Oracle, SQL Server, etc.
