Lógica de Programação


Quando falamos de programação, a Lógica é o principal para você se tornar um grande DEV no futuro. Durante os 4 semestres, os alunos vão aprender desde a construção de algoritmos simples até os mais complexos. Algumas das matérias vistas são: Compreensão e criação de algoritmos, Estruturas de dados, Paradigmas da programação, Operadores e até mesmo a programação orientada a objetos utilizando a linguagem java.
