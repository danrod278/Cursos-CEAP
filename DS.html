Desenvolvimento de Software


A disciplina de Desenvolvimento de Software abrange uma variedade de tópicos que preparam os alunos ara projetar, implementar e manter sistemas de software. Os alunos aprendem desde a introdução até a implementação de manuntenções em softwares extensos. O desenvolvimento de software é uma disciplina em constante evolução, então é importante manter-se atualizado com as tendências, ferramentas e tecnologias emergentes.
